# senior-housing-optimization
**AHP 가중치와 MCLP를 결합한 부산시 시니어주택 최적 입지 선정 프로젝트**

 본 프로젝트는 부산광역시의 고령화와 빈집 문제를 해결하기 위해, 시니어주택 건립에 적합한 최적 입지를 도출하는 데이터 기반 분석 프로젝트이다.

지형적 안전성, 개발 가능성, 빈집 자원, 생활 인프라, 고령인구 수요를 종합적으로 고려하여 후보지를 선별하고, AHP와 MCLP를 결합한 최적화 모델을 통해 최종 입지를 선정한다.

---

## 분석 방법론

### 1. 후보지 필터링
부산시 법정동을 대상으로 시니어주택 건립이 가능한 후보지를 1차 선별한다.
* **빈집 자원**: 구군별 1·2등급 빈집 300개 이상
* **지형 안정성**: 법정동 평균 경사도 15° 이하
* **개발 가능성**: 유효 필지수 500개 이상

<br>

### 2. AHP 기반 입지 적합도 산출
법정동 중심점 기준 반경 1,000m 이내의 생활 인프라 데이터를 활용해 입지 적합도를 계산한다.
각 인프라 밀도를 Min-Max 정규화한 뒤, AHP 가중치를 적용하여 AHP Score를 산출한다.

| 평가 항목 | 세부 지표 | 가중치 |
| :--- | :--- | :--- |
| **의료·보건** | 종합병원 접근성 | 0.46 |
| **대중교통** | 지하철역·버스정류장 밀도 | 0.28 |
| **생활편의** | 대형점포·전통시장 접근성 | 0.17 |
| **노인복지** | 경로당·노인복지관·노인교실 접근성 | 0.09 |

<br>

### 3. MCLP 기반 고령인구 커버리지 분석
`cKDTree` 반경 검색을 활용해 후보지별 1,000m 이내 고령인구를 계산한다. 
이를 통해 시니어주택 설치 시 커버 가능한 부산시 전체 고령인구 비율을 산출한다.

<br>

### 4. AHP-MCLP 결합 최적화
`PuLP`를 활용해 최종 입지 최적화를 수행한다. 
모델은 고령인구 커버율과 입지 인프라 적합도를 동시에 고려하여 최적 입지 조합을 도출한다.

#### [ 목적함수 ]
$$\text{Maximize } \quad (1 - \lambda) \times \text{고령 인구 커버율} \quad + \quad \lambda \times \text{AHP Score}$$

$$\text{Maximize } \quad (1 - \lambda) \sum_{i \in I} \frac{w_i y_i}{\sum_{k \in I} w_k} \quad + \quad \lambda \sum_{j \in J} \frac{\text{AHP}_j x_j}{p}$$

| 기호             | 설명                                                  |
| :------------- | :-------------------------------------------------- |
| $I$            | 전체 수요동 집합                                           |
| $J$            | 후보지 필터링을 통과한 입지 후보동 집합                              |
| $w_i$          | 수요동 $i$의 고령 인구수                                     |
| $\text{AHP}_j$ | 후보지 $j$의 인프라 적합도 점수                                 |
| $x_j$          | 후보지 $j$에 시니어주택을 설치하는지 여부 (Binary)              |
| $y_i$          | 수요동 $i$가 선택된 후보지의 1,000m 반경 내에 포함되는지 여부 (Binary) |
| $p$            | 선정할 시니어주택 입지 개수                                     |
| $\lambda$      | 고령인구 커버율과 인프라 적합도 점수 간 중요도를 조절하는 가중치                |
